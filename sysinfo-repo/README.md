# SysInfo CLI

A lightweight Linux CLI tool to display system information such as OS,kernel,CPU,memory, disk usage, IP address, and current user.

## Features
- Show OS and kernel details
- Display CPU cores
- Show memory and disk usage
- Display IP address and current user
- Supports CLI flags

## Usage

sysinfo --all
sysinfo --cpu
sysinfo --memory
sysinfo --disk
sysinfo --help

## Installation

git clone https://github.com/yourname/sysinfo.git
cd sysinfo
chmod +x sysinfo
sudo cp sysinfo /usr/local/bin/

## Requirements
- Linux
- Bash
- coreutils
